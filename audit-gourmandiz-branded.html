Audit SEO
Problèmes identifiés
❌
Meta description absente
Critique
Aucune balise meta description n'est définie. Google génère un extrait aléatoire, réduisant considérablement le taux de clic dans les résultats de recherche.
📸
87% des images sans attribut alt
Critique
168 images sur 194 n'ont pas de texte alternatif. Impact direct sur le SEO images, l'accessibilité et la compréhension du contenu par Google.
168/194 sans alt 87% des images
🔗
Open Graph incomplet
Important
Pas d'og:image et l'og:description est juste "Gourmandiz". Les partages sur les réseaux sociaux n'afficheront aucun visuel attractif.
📋
Maillage interne faible
Important
133 liens externes contre seulement 53 liens internes. Ce ratio déséquilibré dilue le PageRank et affaiblit le référencement.
53 internes 133 externes
📝
H1 non optimisé pour le SEO
Moyen
Le H1 actuel est orienté concours marketing et ne contient aucun mot-clé SEO lié aux produits (pâte à tartiner, protéinée, sans sucre...).

Plan d'action
Recommandations prioritaires
Priorité haute
1. Ajouter la meta description
Rédiger une meta description de 150-160 caractères incluant les mots-clés principaux : "pâte à tartiner protéinée", "sans sucre", "naturelle".
Impact : +15-25% CTR estimé
Priorité haute
2. Attributs alt sur les 168 images
Ajouter des textes alternatifs descriptifs incluant les noms de produits et mots-clés pertinents sur toutes les images.
Impact : SEO images + Accessibilité
Priorité haute
3. Différer les scripts tiers
Charger Klaviyo, Facebook, Hotjar et GTM en defer/async. Évaluer la nécessité de Chaty + Hotjar simultanément.
Impact : -30% temps de chargement
Priorité moyenne
4. Compléter l'Open Graph
Ajouter une og:image (1200x630px) et une og:description complète pour un meilleur partage sur les réseaux sociaux.
Impact : Engagement social +
Priorité moyenne
5. Corriger les CSS render-blocking
Inliner le CSS critique (above-the-fold) et charger les 10 CSS de composants en différé. Améliore le First Contentful Paint.
Impact : FCP amélioré
Priorité moyenne
6. Renforcer le maillage interne
Ajouter des liens internes vers les collections, le blog et les fiches produit. Objectif : ratio 70% interne / 30% externe.
Impact : Distribution du PageRank
Priorité basse
7. Optimiser le H1
Inclure des mots-clés ciblés dans la structure de titres tout en gardant l'aspect marketing du site.
Impact : Pertinence sémantique
Priorité basse
8. Optimiser le DOM & images
Convertir les images en WebP/AVIF. Lazy-loader les images below-the-fold. Simplifier la structure HTML du body.
Impact : CLS + Vitesse
